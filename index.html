Overview

NatiTube is a code-injection project designed to bypass the limitations of the YouTube mobile web player. It isolates the HTML5 video stream and delivers it directly to the operating system's native media controller.

True Multitasking

By using the native system player, Picture-in-Picture mode integrates perfectly with your mobile OS window manager.

System Integration

Native controls provide access to system-level gestures for volume, brightness, and AirPlay/Cast features.

How It Works

The core logic utilizes a DOM observation loop that monitors m.youtube.com for the <video> element.

Installation Guide

Android Setup

1. Install a UserScript manager (Kiwi Browser, Orion, or InjecThor).

2. Load the natitube-android.user.js script from the repository.

iOS Setup

1. Access the official Shortcut link via the repository.

2. Execute the shortcut from the Safari Share Sheet while on a YouTube page.
