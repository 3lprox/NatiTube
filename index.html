NatiTube

Native player integration for the YouTube mobile web.

Get Started on GitHub

Native Controls

Leverage your system's default volume, brightness, and playback gestures seamlessly.

Picture-in-Picture

Enable true OS-level floating windows for multitasking without app-imposed limits.

Minimalist Design

Bypass proprietary web overlays for a cleaner, faster, and more responsive viewing experience.

Implementation

Android Integration

Utilize the android-version.js script with mobile injectors like InjecThor or Kiwi. It targets the mobile web DOM to isolate the video stream into the system controller.

iOS Solution

Access the dedicated Apple Shortcut via iOS-url. It allows for direct Safari integration through the native share sheet, triggering the iOS media controller.

Experimental Channel

The beta-version directory contains early-access builds with experimental layout adjustments and advanced DOM manipulation techniques.
